PriceNova is a high-performance web application developed for the Semester-4 Python Programming curriculum. It provides a centralized dashboard for tracking product prices across major e-commerce platforms like Amazon and Flipkart, utilizing a modern, responsive UI inspired by the Google Gemini aesthetic.

PriceNova - Price Comparison Application

[Live Demo: pricenova.vercel.app](https://pricenova.vercel.app)

Dynamic Price Tracking: Compares real-time price points against Manufacturer Suggested Retail Prices (MSRP) to provide actionable buying advice.

Market Sentiment Engine: Automatically categorizes assets as "Price Dropping," "Trending Up," or "Stable" using custom Python logic.

Full-Stack Architecture: Built with Flask (Backend), SQLAlchemy (ORM), and SQLite (Database) for robust data management.

Responsive UI: Implemented using Bootstrap 5 and custom CSS, ensuring perfect aspect ratio adaptation for mobile, tablet, and desktop devices.

Cloud Optimized: Pre-configured for serverless deployment on Vercel/Render with integrated CI/CD pipelines.

Tech Stack:

Language: Python 3.12

Framework: Flask

Database: SQLite

Frontend: HTML5, CSS3, JavaScript (ES6+), Bootstrap 5

Deployment: Vercel Configuration (JSON-based serverless functions)
